# 2026-04-22

## 照片筛选工具 (photo_culler.py) - 添加多选批量操作功能

### 改动内容
1. **Listbox 启用多选**：`selectmode=tk.EXTENDED`，支持 Shift 范围选、Ctrl 逐项选
2. **新增批量操作 UI**：在操作按钮下方增加批量操作行（批量保留/标记删除/跳过/恢复），多选时自动激活
3. **快捷键兼容多选**：Enter/Del/S/Z 在多选时自动作用于所有选中项，新增 Ctrl+A 全选
4. **多选预览**：多选时预览区显示选中数量和状态统计，而非单张预览
5. **批量后推进**：批量操作完成后自动定位到下一个未删除的照片
6. **辅助方法**：`_get_selected_indices`、`_mark_entries`、`_advance_after_batch`、`_show_multi_selection`、`_update_multi_selection_label`、`_shortcut_select_all`

### 向后兼容
- 单选行为完全不变
- 现有快捷键、状态持久化、右键菜单均不受影响
